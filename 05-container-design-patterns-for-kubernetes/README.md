# Overview

## Sidecar pattern

顾名思义，Sidecar 指的就是我们可以在一个 Pod 中，启动一个或多个辅助容器，来完成一些独立于主进程（主容器）之外的工作。

它主要利用在同一 Pod 中的容器可以共享存储空间的能力。
